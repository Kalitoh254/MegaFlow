**Tagline:** *Healing in Music*

---

## Overview

MegaFlow Music Inc. is a pioneering music platform dedicated to empowering African artists and delivering authentic Hip-Hop and Afrobeat experiences. Our mission is to heal, inspire, and connect through music while nurturing upcoming talent and creating a global community for music lovers.

---

## Features

- **Responsive Website**: Optimized for desktop and mobile with dark theme aesthetics.  
- **Splash Screen**: Elegant animated splash introducing the brand.  
- **Development Banner**: Notifies visitors that the site is currently in active development.  
- **Floating Band Members**: Interactive carousel with drag & auto-slide functionality.  
- **Hero Section**: Engaging welcome message with background imagery.  
- **About & Vision Sections**: Introduces the band, CEO, and core vision.  
- **Music Section**: Displays latest tracks with playable buttons.  
- **Events Section**: Showcases upcoming concerts, tours, and launches.  
- **Subscription Form**: Collects emails for updates.  
- **Social Follow Section**: Links to YouTube, Instagram, Facebook, and Twitter.  
- **Sticky Floating Home Button**: Quick navigation back to home.  

---

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6)  
- **Responsive Design**: Mobile-first layout with flexible components  
- **Animations & Interactivity**: CSS animations, JS scroll effects, drag-enabled carousels  
- **Hosting**: GitHub repository + Netlify deployment
